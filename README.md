## 📝 Sobre o Projeto
Esta é uma Landing Page Interativa que permite aos usuários explorar os principais guerreiros da franquia Dragon Ball Z. O projeto foi construído do zero, aplicando conceitos modernos de desenvolvimento Front-end para criar uma interface imersiva, com sons, animações e troca de conteúdo em tempo real.

### 🌟 Funcionalidades Principais:
* Invocação de Personagens: Clique nas Esferas do Dragão para carregar instantaneamente as informações e imagens de cada guerreiro.

* Trilha Sonora Épica: Sistema de áudio integrado com controles de Play/Pause para aumentar a imersão.

* Design "Mobile-First": Totalmente otimizado para smartphones, tablets e desktops.

* Animações Dinâmicas: Efeitos de flutuação nos personagens e zoom responsivo nas esferas ao interagir.

---

## 🛠️ Tecnologias Utilizadas
A pilha de tecnologia escolhida para este projeto foca na performance e na fidelidade visual:

* HTML5: Estrutura semântica dos elementos.

* CSS3: * Flexbox & Grid: Organização do layout.

* Keyframes: Animações de flutuação e estados ativos.

* Media Queries: Responsividade extrema para telas de 400px a 979px.

* JavaScript: * Manipulação de eventos (Click, Play/Pause).

* Lógica baseada em Objetos de Dados para gerenciar o conteúdo de forma escalável.
